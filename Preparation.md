## Data
The $data_root is set to be `./datasets` in our default configuration.
```bash
$data_root
|--- imagenet
│   |--- train/
│   │   ├── n01440764/
│   │   │   ├── n01440764_10026.JPEG
│   │   │   └── ...
│   │   ├── n01443537/

│   │   │   └── ...
│   │   ├── ...
│   │   └──
│   └── val/
│   │   ├── n01440764/
│   │   │   ├── ILSVRC2012_val_00000946.JPEG
│   │   │   └── ...
│   │   ├── n01443537/
│   │   │   ├── ILSVRC2012_val_00001269.JPEG
│   │   │   └── ...
│   │   ├── ...
│   │   └──
```
